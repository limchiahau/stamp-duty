# Usage

	python3 main.py rental_amount tenure

replace *rental_amount* and *tenure* with the relevant values.

# Example Output

Given the rental amount of RM1500 and a tenure period of 1 year:

	python3 main.py 1500 1

 The output is:

	Stamp Duty      : RM63
	Copy Amount     : RM10
	Total           : RM73
